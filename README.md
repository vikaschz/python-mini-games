# Python Mini Games Collection 🕹️🎮🎲

A collection of simple terminal-based Python games and utilities built while learning core programming concepts.


## Overview

Each program in this repository is designed to be lightweight and beginner-friendly.
All games run in the terminal and demonstrate practical usage of Python concepts like loops, conditionals, and randomness.

---

## Project Structure

### Games

* **snake_water_gun.py**
  A command-line implementation of the classic Snake-Water-Gun game (similar to Rock-Paper-Scissors). Demonstrates conditional logic and random number generation.

* **rock paper scissors.py**
  Standard Rock-Paper-Scissors game against the computer. Focuses on user input handling and game logic.

* **number guessing game.py**
  The computer generates a random number, and the player attempts to guess it. Reinforces loops, conditions, and input validation.

* **tic_tac_toe.py**
  Two-player Tic-Tac-Toe game played in the terminal. Demonstrates board representation and game state management.

* **hangman.py**
  Word guessing game where the player tries to guess letters within limited attempts. Covers string manipulation and loops.

* **quiz.py**
  A simple quiz game that asks questions and evaluates user responses. Focuses on control flow and scoring logic.

* **ludo.py**
  A basic simulation of Ludo mechanics using randomness. Demonstrates dice rolling and game progression.

* **snake_ladder_game.py**
  Snake and Ladder game simulation. Shows how to manage positions and random movement.

* **slot machine.py**
  A slot machine simulation using random values. Demonstrates probability and basic game mechanics.

---

### Utilities

* **Dice_rolling_simulator.py**
  Simulates rolling a dice and displays random outcomes. Useful for understanding randomness.

* **alarmClock.py**
  A simple alarm clock program that triggers at a specified time. Demonstrates time handling and loops.

* **raw.py**
  Contains basic or experimental Python code snippets used during learning.

---

### Other Files

* **audio.wav**

* **README.md**

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/python-mini-games.git
   cd python-mini-games
   ```

2. Run any Python file:

   ```bash
   python filename.py
   ```
